## (SuperOffice Only)

Configuration values related to Suspended state.

```xml
<SuspendedSite>
  <add key="suspended" value="" />
  <add key="download_baseurl" value="" />
</SuspendedSite>
```

|Name|Description|
|------------|----|
|suspended|The current state of the site.|
|download_baseurl|Download base url.|
